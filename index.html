ประเด็นเปรียบเทียบ SaaS PaaS IaaS
ชื่อเต็ม Software as a Service Platform as a Service Infrastructure as a Service
รูปแบบการให้บริการ ผู้ให้บริการจัดให้ทุกอย่าง ตั้งแต่โครงสร้างพื้นฐานจนถึงซอฟต์แวร์ ผู้ให้บริการจัดเตรียม Platform และ Runtime สำหรับรัน Code ผู้ให้บริการจัดเตรียมทรัพยากรพื้นฐาน เช่น Server, Storage, Network
สิ่งที่ผู้ใช้รับผิดชอบ จัดการแค่ ข้อมูล (Data) และการตั้งค่าผู้ใช้ จัดการ แอปพลิเคชัน และฐานข้อมูลที่พัฒนาขึ้น จัดการ OS, Middleware, Runtime และการติดตั้งแอปฯ ทั้งหมด
ตัวอย่างจริง Gmail, Microsoft 365, Zoom Google App Engine, Heroku AWS EC2, Azure VM, GCE
